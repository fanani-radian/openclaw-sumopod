# Contributing to OpenClaw Sumopod

Terima kasih mau berkontribusi! 🎉

## Cara Berkontribusi

### 1. Fork & Clone
```bash
git clone https://github.com/fanani-radian/openclaw-sumopod.git
cd openclaw-sumopod
```

### 2. Buat Branch
```bash
git checkout -b feature/nama-fitur
# atau
git checkout -b fix/nama-bug
```

### 3. Commit
```bash
git add .
git commit -m "Add: deskripsi singkat"
```

Format commit message:
- `Add:` untuk fitur baru
- `Fix:` untuk bug fix
- `Update:` untuk update konten
- `Docs:` untuk dokumentasi

### 4. Push & PR
```bash
git push origin feature/nama-fitur
```

Lalu buat Pull Request di GitHub.

---

## Konten yang Diterima

### ✅ YES
- Use cases dari pengalaman nyata
- Tips & tricks yang teruji
- Konfigurasi (tanpa secrets!)
- FAQ dan jawabannya
- Video tutorials
- Translations

### ❌ NO
- API keys atau credentials
- Spam atau self-promotion
- Konten berbahaya/illegal
- Plagiarism

---

## Review Process

1. Submit PR
2. Maintainer review (1-3 hari)
3. Feedback (jika ada)
4. Merge 🎉

---

## Code of Conduct

- Hormati sesama kontributor
- Konstruktif dalam feedback
- Fokus pada konten yang bermanfaat

---

Buka [issue](../../issues/new) atau tanya di Discord.
